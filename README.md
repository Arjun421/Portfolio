# Arjun Singh — Portfolio

A modern, animated developer portfolio built with Next.js, Tailwind CSS, and Framer Motion. Features a 3D Three.js background, smooth scroll animations, and a fully responsive design.

## Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Styling:** Tailwind CSS, shadcn/ui
- **Animations:** Framer Motion, Three.js
- **Language:** TypeScript
- **Contact Form:** Formspree
- **Deployment:** Vercel

## Sections

- **Hero** — Name, tagline, resume download
- **About** — Bio, tech arsenal
- **Skills** — Categorized technical skills (Frontend, Backend, AI/ML, DSA, Tools)
- **Projects** — Featured projects with 3D card hover and modal detail view
- **Experience** — Work experience timeline
- **Achievements** — Animated stats counter
- **Contact** — Contact form (Formspree) + social links

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
├── app/              # Next.js app router
├── components/
│   ├── canvas/       # Three.js scene
│   ├── layout/       # Navbar, Footer
│   ├── sections/     # Page sections
│   └── ui/           # shadcn/ui components
├── lib/              # Utilities
public/               # Static assets, resume PDF
```
